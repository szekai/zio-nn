zio-nn

Neural Network Library for ZIO — write once, run on any JVM deep learning framework.

Scala 3.8.3 ZIO 2.1 v0.5.4 GitHub →

Install

// build.sbt
libraryDependencies += "io.github.szekai" %% "zio-nn-djl" % "0.5.4"
// or for DL4J backend:
// libraryDependencies += "io.github.szekai" %% "zio-nn-dl4j" % "0.5.4"

Quick Start

import zio.nn.dsl.*
import zio.nn.*

// Define architecture
val arch = Sequential(7)(LSTM(64, Tanh), Dense(32, ReLU), Output(1, MSE)).build

// Create model
val model = ZModel.create(arch, "my-model").get

// Predict
val predictions = model.predict(features).get

// Train
model.fit(features, labels, epochs = 50)
model.close()

Swap DJL ↔ DL4J

Change one line in build.sbt — zero code changes:

"io.github.szekai" %% "zio-nn-djl"  % "0.5.4"  // ← swap this
"io.github.szekai" %% "zio-nn-dl4j" % "0.5.4"  // ← to this

Features

🔌 Unified API

predict(), fit(), create() — identical across all backends

🧱 Architecture DSL

Sequential, LSTM, Dense, Conv2D, Output — auto-propagating dimensions

🔢 Tensor Ops

add, matMul, transpose, sum, mean — 17 operations with ZIO

🪝 Escape Hatches

.underlying gives raw framework access for multi-GPU, Spark, custom layers

📐 ZIO-Native

zioApi with Scope-based resource management

🔄 Implicit Conversions

Array ↔ NDArray/INDArray bridges for native interop
